QuietSpace is a Streamlit web application developed to analyze how background noise affects focus and mental fatigue.

## Problem Statement

Background noise can reduce concentration and increase mental fatigue, especially for students and people working in shared environments.

## Solution

This application helps users understand the possible impact of different noise levels on focus and suggests suitable actions to create a better study or work environment.

## Features

* Select background noise level
* Analyze focus level
* Estimate mental fatigue
* Get personalized recommendations
* Simple and user-friendly Streamlit interface

## Technologies Used

* Python
* Streamlit
* Prompt Engineering
* GitHub

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/manishadharani1305-glitch/QuietSpace-Noise-Focus-Analysis.git
```

2. Install required packages:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
